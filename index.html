Purpose

This repository contains an introductory R and qtl workshop. While this project is a fork of a LaTeX style, that style has not really been used because it could not be made to work with knitR. The significant new content is the coursework itself which has mostly been written by Angus Wallace and Julian Taylor with help from Nathan Watson-Haigh.
